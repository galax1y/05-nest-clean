# API project w/ [NestJS](https://github.com/nestjs/nest) Framework.

## Description

Built with TypeScript, Domain Driven Development, Clean Architecture & Clean Code in mind.

## Main libs/techs:

- NestJS
- PostgreSQL
- Prisma
- Zod
- Vitest
- Docker

## Installation

```bash
$ npm install
```

## Running the app

```bash
# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
$ npm run test:e2e:watch
```
